Setup

The setup for this website included:

  • Built using Jekyll
  • Based on Poole with the Hyde theme
  • Version control managed by Git and Github
  • Hosted through Github Pages
  • Research and data gathering done through Evernote, the backup brain
  • Heavily developed using Vim, the text editor that works
  • Citations managed by Mendeley
